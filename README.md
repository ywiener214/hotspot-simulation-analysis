# Molecular Dynamics Analysis

This repository contains data and scripts used for analysis in The importance of structural dynamics in genomic hotspots: An in-silico understanding of the KRAS 12 mutation at the base-pair level .

## Contents
- `data/` – Expanded hotspot database, RMSD, RMSF, and structural files
- `scripts/` – Python/R scripts for analysis
- `parameters/` – Parameters used for MD simulations

## Usage
Scripts can be run using standard Python/R environments. See individual scripts for details.

## Data Availability
This repository is archived at Zenodo (DOI: 10.5281/zenodo.19652704).
